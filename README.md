# VPN Server Setup

Step-by-step guide to set up your own VPN server.

## Methods

- 🟢 **WireGuard** - Modern, fast, secure
- 🟡 **OpenVPN** - Classic, well-supported
- 🔵 **WireGuard** on Raspberry Pi

## Quick Setup

```bash
curl -s https://get.vpnsetup.com | sudo bash
```

## Requirements

- VPS or Raspberry Pi
- Ubuntu/Debian
